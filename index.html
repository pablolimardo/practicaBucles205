MISIÓN FOR

Recorrer Listas con FOR

La Línea de Ensamblaje

Imaginá una cinta transportadora en una fábrica. Cada pieza pasa por el mismo proceso, una por una. El bucle for es como esa cinta: procesa cada elemento de la lista automáticamente.

Python 
# BÁSICO: Recorrer una lista
temperaturas = [22.5, 23.1, 24.2, 23.8, 22.9]

for temp in temperaturas:
    print(f"Temperatura: {temp}°C")

# SALIDA:
# Temperatura: 22.5°C
# Temperatura: 23.1°C
# ...
Python 
# INTERMEDIO: Con índice usando enumerate()
sensores = ["OK", "ALERTA", "OK", "CRITICO"]

for i, estado in enumerate(sensores):
    print(f"Sensor {i+1}: {estado}")

# SALIDA:
# Sensor 1: OK
# Sensor 2: ALERTA
# ...
Python 
# AVANZADO: Filtrar y calcular
lecturas = [45.2, 52.1, 48.5, 55.8, 47.3]
LIMITE = 50.0

alertas = []
for valor in lecturas:
    if valor > LIMITE:
        alertas.append(valor)

print(f"Valores sobre límite: {len(alertas)}")
print(f"Alertas: {alertas}")

MISIÓN WHILE

Bucles con Condición: WHILE

El Vigilante Nocturno

Un bucle while es como un guardia que vigila una condición. Mientras esa condición sea verdadera, sigue trabajando. Cuando cambia, se detiene.

Python 
# BÁSICO: Contador simple
contador = 0

while contador < 5:
    print(f"Ciclo {contador}")
    contador += 1  # IMPORTANTE: incrementar o se repite infinito

# SALIDA:
# Ciclo 0
# Ciclo 1
# ...
# Ciclo 4
Python 
# INTERMEDIO: Simulación de llenado de tanque
nivel = 0
CAPACIDAD = 100

while nivel < CAPACIDAD:
    print(f"Nivel actual: {nivel}%")
    nivel += 10  # Aumenta 10% cada ciclo

print("¡Tanque lleno!")
Python 
# AVANZADO: Validación de entrada
password_correcta = "ACPI2026"
intentos = 0
MAX_INTENTOS = 3

while intentos < MAX_INTENTOS:
    password = input("Ingresá la contraseña: ")
    
    if password == password_correcta:
        print("¡Acceso concedido!")
        break  # Sale del while antes de tiempo
    else:
        intentos += 1
        print(f"Incorrecto. Te quedan {MAX_INTENTOS - intentos} intentos")

if intentos == MAX_INTENTOS:
    print("Sistema bloqueado")
!
CUIDADO: Si olvidás incrementar la variable de control, el bucle será infinito. Siempre verificá que la condición pueda volverse falsa.

MISIÓN LISTAS

Métodos Fundamentales de Listas

Python 
# OPERACIONES BÁSICAS
sensores = ["temp", "presion", "caudal"]

# Agregar al final
sensores.append("ph")
print(sensores)  # ["temp", "presion", "caudal", "ph"]

# Sacar del final
ultimo = sensores.pop()
print(ultimo)    # "ph"
print(sensores)  # ["temp", "presion", "caudal"]

# Sacar por posición
primero = sensores.pop(0)
print(primero)   # "temp"

# Tamaño de la lista
print(len(sensores))  # 2
Python 
# BÚSQUEDA Y CONTEO
valores = [10, 20, 10, 30, 10, 40]

# Contar apariciones
print(valores.count(10))  # 3

# Buscar posición (primera aparición)
print(valores.index(30))  # 3

# Verificar si existe
if 50 in valores:
    print("Encontrado")
else:
    print("No existe")  # Esta línea se ejecuta
Python 
# COLA FIFO (First In, First Out)
cola = []

# Agregar al final (llegan clientes)
cola.append("Cliente 1")
cola.append("Cliente 2")
cola.append("Cliente 3")
print(f"Cola: {cola}")

# Atender desde el principio
atendiendo = cola.pop(0)
print(f"Atendiendo: {atendiendo}")  # Cliente 1
print(f"Cola restante: {cola}")     # ["Cliente 2", "Cliente 3"]

Monitor de Nivel con LCD

Potenciómetro + LCD I2C + Arduino

Tu Misión Final

Crear un sistema que lea un potenciómetro (simulando un sensor de nivel) y muestre en tiempo real el porcentaje en una pantalla LCD I2C. Vas a combinar hardware y programación.

Componentes Necesarios

🔲

Microcontrolador

📟

Pantalla

🎚

Potenciómetro

10kΩ

🔌

Cables

Conexiones

Paso a Paso

  1. Armado del circuito: Conectá el LCD I2C a los pines SDA y SCL del Arduino. Conectá el potenciómetro al pin analógico A0.
  2. Configuración del LCD: Importá la biblioteca LiquidCrystal_I2C y configurá la dirección (generalmente 0x27 o 0x3F).
  3. Lectura del sensor: Usá analogRead(A0) para leer el potenciómetro (valores de 0 a 1023).
  4. Conversión a porcentaje: Mapeá el valor de 0-1023 a 0-100 usando map().
  5. Mostrar en LCD: Actualizá la pantalla cada 500ms mostrando "Nivel: XX%".
Arduino C++ 
// Código base para arrancar
#include <Wire.h>

// Inicializar LCD (dirección, columnas, filas)

  lcd.init();
  lcd.backlight();
  lcd.setCursor(0, 0);
  lcd.print("Monitor Nivel");
  delay(2000);
  lcd.clear();
}
